## Установка

```bash
npm install
```

## Запуск приложения

```bash
# режим разработки
npm run start

# режим автоматического обновления
npm run start:dev

# производственный режим
npm run start:prod
```

## Тестирование

```bash
# модульные тесты
npm run test

# сквозные тесты
npm run test:e2e

# покрытие тестами
npm run test:cov
```
